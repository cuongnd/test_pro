AliceJS Demos

AliceJS - (A Lightweight Independent CSS Engine) is a micro JavaScript library focused on using hardware-accelerated capabilities (in particular CSS3 features) in modern browsers for generating high-quality, high-end visual effects.

AliceJS Demos

Please browse through our collection of demos of the Cheshire and Caterpillar plugins, or try developing one of your own plugins.

Cheshire

Cheshire is a simple to use yet remarkably expandable engine for building and implementing CSS3 based animations.

Available Effects:

  • bounce()
  • dance()
  • drain()
  • fade()
  • hinge()
  • pageFlip()
  • pendulum()
  • phantomZone()
  • raceFlag()
  • slide()
  • spin()
  • toss()
  • twirl()
  • wobble()
  • zoom()
Caterpillar

Caterpillar is a fully customizable 3D rendering that enables interaction with divs enabling the client to go further than simple animations. Stepping into a position to push the limits of human computer interaction.

Available Effects:

  • book()
